초대 없이 가입 가능, 승인 전에는 Trial

GitHub Pages에서 바로 미리볼 수 있는 정적 프런트엔드 세트입니다. 실제 여러 기기 동기화는 Supabase 백엔드와 연결되어 있습니다.

서비스 개념

유료/무료 서비스를 한눈에 파악할 수 있는 개인용 관리 보드입니다. 누구나 가입하고 기능을 써볼 수 있지만, 관리자 승인 전에는 Trial 계정으로 간주되어 데이터가 언제든 삭제될 수 있습니다.

Trial 즉시 사용 승인 후 계속 사용 관리자는 전체 열람 가능

공통 규칙

보안상 ID/PASS 필드는 실제 값을 넣는것을 추천하지 않습니다. 위장 규칙에 맞게 표시되므로 기억할수 있는 변칙값이 보여집니다

  • 일반 ID: 첫 글자/끝 글자 유지, 중간 랜덤, 길이 동일
  • 이메일: 도메인 유지, 앞부분만 위장
  • 전화번호: 010 유지, 마지막 숫자 유지, 중간 랜덤
  • PASS: 첫/끝 문자처럼 보이는 것까지 모두 가짜

1. Sign Up

누구나 가입 가능합니다. 가입 직후 바로 로그인해서 기능을 써볼 수 있습니다.

2. Trial Use

관리자 승인 전에는 Trial 계정입니다. 기능은 써볼 수 있지만 데이터 보존은 보장되지 않습니다.

3. Admin Approval

관리자는 전체 사용자와 데이터를 보고, 승인·비활성화·삭제·정원 제한을 관리합니다.

데모 안내

Supabase 백엔드와 연결되어 있어 여러 기기에서 동기화됩니다. 기본 관리자 계정으로 Admin 페이지에서 사용자를 관리할 수 있습니다.

Private Vault Board - Personal URL Organizer

GitHub Pages에서 바로 미리볼 수 있는 정적 프런트엔드 세트입니다. 실제 여러 기기 동기화는 Supabase 백엔드와 연결되어 있습니다.

서비스 개념

유료/무료 URL 소팅 관리를 위한 개인용 관리 보드입니다. 누구나 가입하고 기능을 써볼 수 있지만, 관리자 승인 전에는 Trial 계정으로 간주되어 데이터가 언제든 삭제될 수 있습니다

Trial 즉시 사용 승인 후 계속 사용 관리자는 전체 열람 가능

공통 규칙

ID/PASS 필드는 실제 값이 아니라 유추를 위한 값입니다. 일반 ID, 이메일, 전화번호, PASS 모두 변칙 규칙에 맞게만 저장합니다.

  • 초대 없이 가입 가능, 승인 전에는 Trial
  • ID/PASS는 변칙규칙이 적용되어 표기됨
  • ID/PASS 는 저장되지 않음
  • 개인정보는 보호되지 않습니다

1. Sign Up

누구나 가입 가능합니다. 가입 직후 바로 TRIAL 로그인해서 기능을 써볼 수 있습니다.

2. Trial Use

관리자 승인 전에는 Trial 계정입니다. 기능은 써볼 수 있지만 데이터 보존은 보장되지 않습니다.

3. Admin Approval

관리자는 전체 사용자와 데이터를 보고, 승인·비활성화·삭제·정원 제한을 관리합니다.

데모 안내

Supabase 백엔드와 연결되어 있어 여러 기기에서 동기화됩니다. 기본 관리자 계정으로 Admin 페이지에서 사용자를 관리할 수 있습니다.

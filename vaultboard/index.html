초대 없이 가입 가능, 승인 전에는 Trial

GitHub Pages에서 바로 미리볼 수 있는 정적 프런트엔드 세트입니다. 실제 여러 기기 동기화는 Supabase 백엔드와 연결되어 있습니다.

바로 가입 로그인

서비스 개념

가짜 ID/PASS만 저장하는 개인용 관리 보드입니다. 누구나 가입하고 기능을 써볼 수 있지만, 관리자 승인 전에는 Trial 계정으로 간주되어 데이터가 언제든 삭제될 수 있습니다.

Trial 즉시 사용 승인 후 계속 사용 관리자는 전체 열람 가능

공통 규칙

ID/PASS 필드는 실제 값이 아니라 가짜 기록용 값입니다. 일반 ID, 이메일, 전화번호, PASS 모두 위장 규칙에 맞게만 저장합니다.

  • 일반 ID: 첫 글자/끝 글자 유지, 중간 랜덤, 길이 동일
  • 이메일: 도메인 유지, 앞부분만 위장
  • 전화번호: 010 유지, 마지막 숫자 유지, 중간 랜덤
  • PASS: 첫/끝 문자처럼 보이는 것까지 모두 가짜

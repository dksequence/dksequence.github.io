Trial 정책 안내

이 서비스는 가입 즉시 써볼 수 있지만, Trial 계정은 정식 승인 전까지 데이터 보존이 보장되지 않습니다.

Trial에서 가능한 것

  • 로그인 가능
  • 카테고리 추가/수정/삭제
  • 서비스 추가/수정/삭제
  • 가짜 ID / PASS 기록 테스트

Trial에서 보장되지 않는 것

  • 데이터 영구 보존
  • 관리자 승인 전 장기 사용
  • 삭제된 데이터 복구

정식 승인 기준

관리자는 전체 사용자와 데이터를 열람할 수 있으며, Trial 사용자를 Active 상태로 승인하거나, 필요 시 데이터를 포함해 삭제할 수 있습니다. 관리자 정식 승인 가능한 활성 사용자 수는 별도 정원으로 제한할 수 있습니다.

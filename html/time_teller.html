Time Teller — Voice-based time notification system

A minimal time-based interaction tool designed to announce the current time at defined intervals. The interface focuses on clarity, rhythm, and presence, allowing users to experience time not as numbers, but as a continuous auditory sequence.

Designed with a dark minimal UI and structured layout, this application bridges utility and emotional awareness through voice interaction. The right-side section works as a presentation area, pairing the Android application capture with a concise project description so the desktop layout feels more complete and balanced against the live mobile mockup on the left.
